# CSI 特别兴趣小组

k8s 和 CSI 协议在不断的演进，社区的 CSI 还有一些潜在的问题和很多需要改进的地方，SIG CSI 负责对这些相关的问题进行改进，让社区的 k8s 支持更上一个档次。

[章程](charter.md) 定义了 CSI 特别兴趣小组的范围和治理。
